## 多级缓存的使用

#### maven引入
```
  <dependency>
        <groupId>com.y3tu</groupId>
        <artifactId>y3tu-tool-cache</artifactId>
        <version>2.0</version>
  </dependency>
```

#### 多级缓存配置

* **y3tu-tool-cache.servlet-enable** _是否开启servlet页面访问_
* **y3tu-tool-cache.stats**  _是否开启缓存统计_
* **y3tu-tool-cache.url-pattern**  _配置拦截的url_
